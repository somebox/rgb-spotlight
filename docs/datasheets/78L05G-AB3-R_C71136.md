# UTC 78L05G-AB3-R — Technical Summary

## Overview
3-terminal, fixed **+5V** positive linear voltage regulator in **SOT-89-3** package (Halogen Free / Lead Free). Part of the 78LXX family.

---

## Key Specifications

| Parameter | Value |
|-----------|-------|
| Output Voltage (V_OUT) | **5.0V** (4.80V min / 5.20V max @ 25°C) |
| Input Voltage Range (V_IN) | **7V – 20V** (operating); 30V absolute max |
| Output Current (I_OUT) | **100mA** max |
| Dropout Voltage (V_D) | **1.7V** typ @ 25°C, 40mA |
| Quiescent Current (I_Q) | **2.0mA** typ / 5.5mA max |
| Ripple Rejection | **80dB** typ @ 120Hz (41dB min) |
| Output Noise Voltage | **40µV** typ (10Hz–100kHz) |
| Load Regulation | 15mV typ / 60mV max (1–100mA) |
| Line Regulation | 8mV typ / 150mV max (7–20V) |
| Temp. Coefficient | −0.65 mV/°C typ |
| Operating Temperature | **−40°C to +85°C** |
| Junction Temperature | +150°C max |
| Power Dissipation (SOT-89) | **350mW** max |

---

## Pin Descriptions — SOT-89-3 (AB3 variant)

> Ordering suffix **-AB3-R** pin assignment: **O – G – I**

| Pin # | Name | Function |
|-------|------|----------|
| **1** | OUTPUT (O) | Regulated 5V output |
| **2** | GND (G) | Ground reference |
| **3** | INPUT (I) | Unregulated DC input |

> ⚠️ Note: The **-AB3-C-R** variant has reversed pinout (G–I–O). Verify suffix carefully.

## Typical Application Circuit

```
        C1 (0.33µF)          C0 (0.1µF)
V_IN ──┬──────────────[Pin 3]──[78L05]──[Pin 1]──┬── V_OUT (+5V)
       │                      [Pin 2]             │
      GND ──────────────────────┴─────────────────┴── GND
```

| Component | Value | Purpose |
|-----------|-------|---------|
| C1 (input cap) | **0.33µF** | Input bypass / stability |
| C0 (output cap) | **0.1µF** | Output bypass / transient response |

> Place capacitors as close as possible to the device pins.

## Protection Features
- **Over-current protection** (current limiting)
- **Short-circuit protection**
- **Thermal shutdown** (activates before T_J = 150°C)

---

## Important Design Notes

1. **Minimum input voltage:** V_IN must exceed V_OUT + 1.7V dropout → **minimum ~6.7V** for guaranteed regulation.
2. **Power dissipation:** P_D = (V_IN − V_OUT) × I_OUT must not exceed **350mW** for SOT-89 at 25°C; derate above 25°C.
3. **Quiescent current:** Up to **5.5mA** flows through GND pin — account for this in low-power designs.
4. **Effective output current** is limited by thermal dissipation, PCB copper area, and input-output differential; do not rely solely on the 100mA rating without thermal analysis.
5. **Not recommended** for life-support or safety-critical systems per UTC policy.
6. Capacitor ESR affects stability; use standard ceramic or film capacitors at the specified values.
