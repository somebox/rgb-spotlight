| Pin # | Name | Description |
|-------|------|-------------|
| 1 | SW | Drain of internal power switch (NFET); connects to inductor/Schottky diode node |
| 2 | GND | Power and signal ground |
| 3 | DIM | Multipurpose: ON/OFF control, analog dimming (0.5–2.5V DC), PWM dimming input; floats to internal 5V via ~200kΩ pull-up |
| 4 | CSN | High-side current sense input; sampling resistor R_S placed between VIN and CSN |
| 5 | VIN | Supply input (6–30V); must have low-ESR bypass capacitor placed close to pin |
| — | Exposed PAD | Thermal pad; internally connected to GND; must be soldered to PCB ground copper for heat dissipation |

## LED Output Current Formula

$$I_{OUT} = \frac{0.1}{R_S} \quad (R_S \geq 0.082\,\Omega)$$

> R_S is connected between VIN and CSN (high-side sense).

**Analog dimming (0.5V ≤ V_DIM ≤ 2.5V):**
$$I_{OUT} = \frac{0.1 \times V_{DIM}}{2.5 \times R_S}$$

**PWM dimming (V_PULSE > 2.5V):**
$$I_{OUT} = \frac{0.1 \times D}{R_S}$$

### Inductance Selection Guide

| Output Current | Recommended Inductance |
|---------------|----------------------|
| I_OUT > 1A | 27–33 µH |
| 0.8A < I_OUT ≤ 1A | 33–47 µH |
| 0.4A < I_OUT ≤ 0.8A | 47–68 µH |
| I_OUT ≤ 0.4A | 68–100 µH |

> Inductor saturation current must be **1.3–1.5× I_OUT**.

## DIM Pin Operating Modes

| V_DIM Condition | Result |
|----------------|--------|
| < 0.3V | Shutdown; I_OUT = 0; I_Q = 60 µA |
| 0.5V – 2.5V (DC) | Analog dimming; proportional current reduction |
| 2.5V – 5.0V (DC) | Full current (100%); I_OUT = 0.1/R_S |
| PWM, V_high > 2.5V | PWM dimming; I_OUT proportional to duty cycle |
| Floating | Full current (internal 5V via 200kΩ pull-up) |
| PWM @ 100Hz | Dimming ratio up to **5000:1** (min duty 0.02%) |
| PWM @ 20kHz | Dimming ratio up to **25:1** (min duty 4%) |

## Important Design Notes & Limitations

- **High-side current sensing:** R_S connects between **VIN and CSN** (not ground-side). Average sense voltage = 100mV (±5mV).
- **No output capacitor required:** CCM buck topology; output is directly the LED string — do **not** add large output capacitance across LEDs.
- **Schottky diode mandatory:** A freewheeling Schottky diode must connect from SW to VIN (cathode to VIN) to conduct inductor current during switch-off.
- **Soft start:** Connect capacitor from DIM to GND; ~0.8ms per nF ramp time.
- **Bypass capacitor placement:** Must be physically close to VIN pin; minimum 4.7 µF for DC input; larger for rectified AC.
- **PCB layout:** Use star-ground topology. Bond exposed PAD to large GND copper pour. High-current loop (VIN → R_S → LED → L → SW → D → VIN) must be kept tight.
- **Thermal derating:** P_D(max) = (150°C − T_A) / 45°C/W. At T_A = 85°C, max P_D ≈ **1.44W**. Exposed PAD soldering is essential for sustained high-current operation.
- **OTP:** At T_J = 160°C the IC shuts down; auto-recovers at 140°C. Does not latch off.
- **Maximum switch current:** 1.5A absolute max; design for ≤1.2A average.
- **DIM voltage limits:** Never exceed 6V on DIM pin; never apply more than 35V to VIN or SW.
